# My Personal Hub — Frontend

Next.js frontend for [My Personal Hub](../README.md).

## Development

```bash
npm install
npm run dev       # http://localhost:3000
npm run lint      # ESLint
npm run build     # Production build
npm test          # Vitest
```

## Environment

Copy `../.env.example` to `.env.local` and set `NEXT_PUBLIC_API_URL`.
