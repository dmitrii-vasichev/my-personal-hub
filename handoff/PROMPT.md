## Если Claude Code делает что-то не так

Используйте эти фразы:
- **«Сверься с мокапом»** — `handoff/mockups/hub-brutalist-v2.html`, покажи скриншот сравнения
- **«Откатись и сначала задай вопросы»** — если видите, что импровизирует
- **«Делай по этапам»** — если пытается сделать всё сразу
- **«Не трогай X»** — если лезет в то, что не в скоупе

---

## Tips

- Делайте **новую ветку на каждый этап**: `feat/redesign-stage-1-tokens`, `feat/redesign-stage-2-today`, и т.д.
- После каждого этапа — визуальный diff через скриншоты мокапа vs вашего приложения
- Если Claude Code делает сразу большой PR — остановите, разбейте на части
